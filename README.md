# qBits - Copilot Demo Application

## Java

### Prerequisites

- Java 17

### Tests

```bash
./gradlew test
```

## TypeScript

### Prerequisites

- Node.js 18
- Yarn 1.22.19

```bash
yarn
```

### Lint

```bash
yarn lint
```

### Tests

```bash
yarn test
```

### Tests in Watch Mode

```bash
yarn test:watch
```

### Check Types

```bash
yarn check:types
```
